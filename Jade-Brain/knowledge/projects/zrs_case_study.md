# Jade @ ZRS Management

**Client**: ZRS Management (Orlando, FL)  
**Project**: LinkOps AIBOX (Jade)  
**Status**: Offline-first AI assistant for property management

## Overview
Jade is an offline-capable DevSecOps & MLOps assistant built specifically for ZRS Management's property management operations in Orlando, Florida. The system runs entirely offline to ensure data privacy and compliance.

## Technical Implementation
- **Fine-tuned Model**: Phi-3 fine-tuned on Google Colab using ZRS policies and housing laws
- **RAG System**: Retrieval-Augmented Generation over tenant/vendor/workflow data
- **RPA Integration**: Robotic Process Automation for onboarding and work-order completion
- **MCP Tools**: Model Control Protocol tools to send emails and reports to appropriate parties automatically

## Key Features
- Offline operation for data security
- Integration with ZRS Management workflows
- Automated compliance reporting
- Tenant and vendor data management
- Work order processing and automation

## Value Proposition
Provides ZRS Management with an AI assistant that understands their specific policies, local housing laws, and operational procedures while maintaining complete data privacy through offline operation.
